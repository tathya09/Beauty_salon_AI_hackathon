📋 Project Overview

GlowCity — AI-Powered Mumbai Salon Marketplace

We built GlowCity entirely using AI-assisted development. Our primary tool was Kiro (Amazon's AI IDE), which generated the full product from a single prompt — including the system architecture, all Next.js pages, Firebase integration, API routes, and property-based tests. Kiro acted as our engineering team, letting us focus on product decisions.

In-product AI (powered by Google Gemini 1.5 Flash):

• Glow AI Chat — conversational booking assistant that understands natural language queries like "bridal makeup in Andheri under ₹3000" and returns matching salons

• AI Style Matcher — upload any hair/beauty inspiration photo and Gemini Vision extracts style tags, then finds Mumbai salons that specialise in that exact look

• Smart Search — natural language search that extracts service keywords using Gemini and matches against the salon database

• AI Promo Copy Generator — salon owners generate Instagram, WhatsApp and website copy in one click using Gemini

Infrastructure stack (all free tier): Next.js 14 + TypeScript, Firebase (Firestore + Auth + Storage), Vercel, Leaflet/OpenStreetMap maps, Razorpay test mode payments.

The entire platform was designed, coded, tested and deployed within 48 hours using AI tools — a real demonstration of how AI multiplies a small team's output.
